# fast-api-python-boiler

Topics that are Covered:
- Creating CRUD API in fast API and SQL Alchemy
- Custom Request Validation
- JWT Authentication and Role Based Authorization
- Alembic
- Docker Integration
- Redis Integration.
- Global exception handler on rest api
